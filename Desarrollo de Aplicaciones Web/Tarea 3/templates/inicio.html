Bienvenido!

Con el fin de promover la solidaridad y tratar de aprovechar los cultivos que las personas hacen en sus casas, evitando pérdidas por exceso de producción, se desarrolló esta aplicación web que permite informar la disponibilidad de frutas o verduras que se regalan. Además, se puede realizar un pedido de donación por si se necesita alguna fruta o verdura.

A continuación, se presenta un mapa que muestra la ubicación de las últimas donaciones y pedidos realizados. El mapa permite visualizar la información básica de cada donación o pedido. Para obtener más detalles, se puede acceder a nuestras pestañas "Ver donaciones" o "Ver pedidos", donde se encontrará una tabla con información más detallada. Si se requiere obtener información adicional sobre una solicitud en particular, basta con hacer clic en una fila específica para acceder a todos los detalles disponibles de esa donación o pedido.
